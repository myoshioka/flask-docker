flask-docker
=========

Overview

## Description

- Sample of a Docker environment using a Flask app.

## Requirement

- Docker & Docker-Compose

## Quick Start

```bash
git clone git@github.com:myoshioka/flask-docker.git
cd flask-docker
docker-compose up -d
```

## Usage

- Access `http://localhost:5001/`
- Login
  - admin / admin
